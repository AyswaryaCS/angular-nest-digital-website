# Angular Nest Digital Website

This is a responsive website built using Angular.

## Features
- Angular Routing
- Responsive Design
- Component-based structure

## Technologies
- Angular
- TypeScript
- HTML
- CSS
- Bootstrap

## How to Run

1. Clone repository  
   git clone https://github.com/AyswaryaCS/Angular-nest-digital.git

2. Go to project folder  
   cd Angular-nest-digital

3. Install dependencies  
   npm install

4. Run project  
   ng serve

5. Open in browser  
    http://localhost:55412/
